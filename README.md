<h1>🌐 Web Development MCQs</h1>
<p>
  This is a simple web based quiz application that allows users to test their knowledge of web development fundamentals. It is built using HTML, CSS and JavaScript.
</p>

<h3>📋 Features:</h3>
<ul>
  <li>Multiple Choice Questions (MCQs)</li>
  <li>Real-time answer validation</li>
  <li>User friendly interface</li>
  <li>Responsive design (basic)</li>
  <li>Color feedback for correct and incorrect answers</li>
</ul>

<h3>🛠️ Technologies Used:</h3>
<ul>
  <li><b>HTML5</b> - Structure of the web page</li>
  <li><b>CSS3</b> - Styling and layout</li>
  <li><b>JavaScript</b> - Interactivity and answer checking</li>
</ul>


<h4>🚀 Getting Started</h4>
<h3>1. Clone the repository</h3>

  <li>git clone https://github.com/your-username/web-development-mcqs.git</li>
</ul>
<h3>2. Open the project</h3>

</ul>
    web-development-mcqs/ <br />
    ├── index.html <br />
    ├── style.css <br />
    └── script.js
<h3>3. Run the quiz</h3>
<ul>
  <li>Just open the index.html file in your favorite browser and start answering the questions!</li>
</ul>
